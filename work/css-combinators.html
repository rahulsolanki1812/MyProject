div ~ p { background-color: darkgrey; }
this is changing color of those divs which are occuring just below the selector which is div.

div > p { color: red; }
this change the color of paragraph elements which are direct child of divs.

div + p { color: yellow; }
this change the color of those paragraph tag which occurs just after the div end tag.
