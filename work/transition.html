This is how to use Transition Effect in css.

Transition On Hover To Change Element Different Properties.

See Code here: 
                        .div1 button {
                            background-color: chocolate;
                            color: white;
                            padding: 15px 40px;
                            border-radius: 0px;
                            box-shadow: 0;
                            transition-property: box-shadow, border-radius, color, background-color;
                            transition-duration: .2s, 1s, .5s, 1s;
                        }
                        .div1 button:hover {
                            background-color: white;
                            border: 1px solid chocolate;
                            border-radius: 30px;
                            box-shadow: 4px 4px 80px black;
                            color: chocolate;
                        }

Transition-Delay On Hover

See Code Here 
                         .div2 button {
                            background-color: yellowgreen;
                            color: white;
                            padding: 15px 40px;
                            border-radius: 0px;
                            font-size: 12px;
                            transition-property: font-size, border-radius, color, background-color;
                            transition-duration: 1s;
                            transition-delay: 2s;
                        }
                        .div2 button:hover {
                            border: 1px solid yellowgreen;
                            color: yellowgreen;
                            background-color: white;
                            border-radius: 15px;
                        }

Tilt The Element Using skew() function

See Code Here 
                        .div3 button {
                            background-color: saddlebrown;
                            padding: 15px 30px;
                            color: white;
                            transition: transform 2s;
                        }

                        .div3 button:hover {
                            transform: skewX(20deg);
                        }

Scale the element using scale(x,y) function

This takes them number by which width and height gets multiple and according to that size increases. Example: width:100px; and scaleX(2). It will be scaled by 200px; same with the height.

Scale(x,y)

See Code Here 
                                .div4-inner1 button:hover {
                                    transform: scale(2,2);
                                }

ScaleX(x)

See Code Here 
                                .div4-inner2 button:hover {
                                    transform: scaleX(2);
                                }

Scalex(y)

See Code Here 
                                .div4-inner3 button:hover {
                                    transform: scaleY(2);
                                }
