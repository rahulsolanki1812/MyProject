This is how to use gradient in background

Background Gradient

The code used to create background gradient is given below:

.div1 { padding:30px; background: linear-gradient(to top,red,yellow); }

See Detail
  1. The parameter in linear-gradient(direction,start color,end color);
  2. You can define color percentage to. linear-gradient(dir,color 10%, color 50%);
